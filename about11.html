About Configuration Migrator



With 16.11.1 and above releases, C9500-32C, C9500-32QC, C9500-48Y4C, C9500-24Y4C (9500 high performance switches) products has change in default port from Layer-2 to Layer-3 to help product management with Cisco-DNAC.This helps all customers with Cisco DNAC-for Day-0 discovery.
This Configuration Migrator is a web based tool that helps IOS-XE customers to upgrade or downgrade the switch configurations from <16.10.1(or earlier releases) to<-->16.11.1(or later releases)
