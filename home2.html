This configuration migration tool will help to migrate the catalyst 9500 high performance switch configuration from 16.9.1/16.10.1 release to 16.11.1 or later release. This tool eliminates lot of manual editing for the customers while migrating to 16.11.1 or later releases.
In 16.10.1 (and prior releases) default port mode is Layer-3 and with 16.11.1 onwards default port mode is Layer-2. Hence while upgrading to 16.11.1, the default start-up configurations will not directly work and commands will generate error. This default port mode change in 16.11. is introduced by cisco in order to support the Day 0 discovery/deployments along with SDA..
